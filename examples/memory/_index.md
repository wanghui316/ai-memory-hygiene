# memory 目录索引（示例）

> memory/ = 当日要点 + 关键决策指针
> 详细内容 → 看对应业务目录
> 最后更新：YYYY-MM-DD

---

## 文件清单（按时间倒序）

| 文件 | 主题 | 详细内容在哪 |
|---|---|---|
| `YYYY-MM-DD.md` | 今日要点 | 业务目录对应文件 |

---

## 当前生效的关键决策（指针）

- **当前规则**：业务目录规则档最新版本
- **当前持仓**：业务目录持仓跟踪最新版本
- **进行中的项目**：`work_records/` 最新
- **股票分析专题**：`work_records/YYYY-MM-DD_XXX.md`

---

## 写入规则（精简原则）

- 当日要点记录 → `memory/YYYY-MM-DD.md`，只写**做了什么/决定了什么 + 指向哪**
- 详细内容、表格、数据 → 写到对应业务目录
- memory 文件本身 ≤ 30 行；超出的就拆出去
- 每周一次（建议周日）：合并/清理过期条目
