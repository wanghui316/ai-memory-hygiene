## 论文信息

- **标题**：TILE 加载机制：一种面向大模型专业化开发的训练数据纵向结构化方法
- **作者**：辉哥
- **核心贡献**：三层索引加载引擎（TILE）—— RIL / DCL / EBL 三层组织 + 指针驱动的两步定位路由
- **核心结论**（相对 v8.0）：
  - 模拟实验：5%–100% 命中率下 TILE 加载 820 tokens（5% 命中）/ 1390 tokens（100% 命中）
  - 真实实验（15 条 query）：V3 模式质量 4.93 vs V0' 2.40 vs V0 1.00（p < 0.001）
  - 边际效率：V0' → V3 = 91 tok/分（4.7 倍于 V0 → V0'）
  - 可扩展性：0.5M–100M tokens 区间 TILE 加载恒为 820 tokens
