## 二、4.3 节真实实验关键数据

**实验设计**：3 加载模式 × 15 真实 query = 45 次回答
- V0：扁平 11 文件夹描述（约 350 tokens）
- V0'：V0 + 每文件夹摘要（约 950 tokens）
- V3：RIL + DCL + 目标文件（约 1,181 tokens）

**核心结论**：
- 平均质量分：V0 = 1.00 / V0' = 2.40 / **V3 = 4.93**
- 边际效率：V0 → V0' = 428 tok/分 / **V0' → V3 = 91 tok/分**（4.7 倍提升）
- 统计检验：Wilcoxon 符号秩检验，**p < 0.001**（V3 vs V0 / V0' 均显著）

**质量分布**：V3 模式 14/15（93%）达到 4 分以上，V0' 仅 2/15（13%）达到 3 分。

## 三、4.4 节可扩展性关键数据

**实验范围**：0.5M – 100M tokens（GB 级逼近）
- 5% 命中率下，TILE 加载量稳定在 **790 tokens 左右**
- 100M 规模下节省比例达 **99.999%**
- 即使 100% 命中率（最坏场景），10M 工作区下 TILE 加载仅占 0.07%
